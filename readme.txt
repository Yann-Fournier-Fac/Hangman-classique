Ce jeu a été crée par Yann, Elisa, Matthieu et Joshua dans le cadre d'un projet en B1 Informatique.

Les règles simples : deviner le mot imposé sans tuer José.
Pour cela, vous pouvez saisir des lettres ou des mots avec 10 erreures possibles.
Si une lettre est fausse, cela vous est compté comme une erreur.
Si le mot que vous entrez est faux, cela est compté comme deux erreures.

Vous pouvez à tout moment quitter le jeu en entrant le mot "stop".
Vous pouvez sauvegarder votre partie si vous le souhaitez et la retrouver dans le menu principal.

2 versions du jeu sont disponibles : Basique et Ascii !
